docker container ls -> displays list of containers running
docker container ls -a -> displays list of containers running and stopped

docker container stop container id -> to stop a container


docker run -p 5000:5000 -d in28min/todo-rest-api-h2:1.0.0.RELEASE - downloads and run image from docker registry in detached mode from console
so on ctrl+C the service don't terminate

docker run -p 5000:5000 -d in28min/todo-rest-api-h2:1.0.0.RELEASE  outputs a container id

docker log -f container id to see logs

docker run -p 5001:5000 -d in28min/todo-rest-api-h2:1.0.0.RELEASE - run another instance of the container on port 5001 but created from the same image

docker image ls /  docker images

stop a container
docker container stop containerId - gracefully shutdown
docker container kill id - just stop
